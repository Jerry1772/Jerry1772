# aid core
## 프로젝트 목적
기존 자체 제품들이 같은 모듈을 사용하도록 코어 모듈들을 재설계 및 구현
<br/><br/>

## 역할
- Data Fetch (소스별), Data Load (타입별), Preprocess, Train, Evaluate,
Inference, Postprocess 모듈 개발
- 기존 제품 3종의 코드 리팩토링
- 시계열 이상감지 모델 연구개발 (Basket-Decoder, Embedding
Clustering)
- MQ 서버/클라이언트 구현
<br/><br/>
